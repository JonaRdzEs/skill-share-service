# Skill Share 🧠✨

A peer-to-peer skill sharing platform where users can teach, learn, and grow together through 1-on-1 sessions.

---

## 🚀 Tech Stack

**Frontend**

* Next.js (App Router)
* TailwindCSS

**Backend**

* Node.js + Express
* TypeScript
* Prisma ORM

**Database**

* PostgreSQL

---

## 📦 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/JonaRdzEs/skill-share-service.git
cd skill-share-service
```

---

### 2. Install dependencies

```bash
npm install
```

---

### 3. Environment Variables

Create a `.env` file, copy the content from `.env.example` and replace the values with yours

---

### 4. Start database
```bash
  docker compose up -d 
```
  

### 5. Generate Prisma

```bash
  npx prisma migrate dev 
```

---

### 6. Run the project

#### Start backend

```bash
npm run dev
```
